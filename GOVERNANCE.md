# Yönetişim / Governance

Bu belge, QoreChain Guides deposunun nasıl yönetildiğini ve topluluk katkılarının nasıl değerlendirildiğini açıklar.

This document explains how the QoreChain Guides repository is managed and how community contributions are evaluated.

---

## İçerik Politikası / Content Policy

Depoya eklenen tüm içerikler aşağıdaki ilkelere uymalıdır:

All content added to the repository must follow these principles:

1. İçerik doğrulanabilir, tarafsız ve kısa olmalıdır. / Content should be verifiable, neutral, and concise.
2. Resmi olmayan iddialar, ödül garantileri veya mainnet varsayımları içermemelidir. / Content should not include unofficial claims, reward guarantees, or mainnet assumptions.
3. Türkçe ve İngilizce sayfalar mümkün olduğunca aynı yapıyı korumalıdır. / Turkish and English pages should maintain the same structure where possible.
4. Teknik komut değişiklikleri resmi kaynak referansı içermelidir. / Technical command changes should include an official source reference.

## Davranış Kuralları / Code of Conduct

Bu depo yapıcı, saygılı ve kapsayıcı bir katkı ortamı hedefler. Teknik anlaşmazlıklar içerik odaklı olmalı, kişisel olmamalıdır. Sorunlu davranışlar Bakıcıya bildirilebilir.

This repository aims for a constructive, respectful, and inclusive contribution environment. Technical disagreements should be content-focused, not personal. Problematic behavior can be reported to the Maintainer.
