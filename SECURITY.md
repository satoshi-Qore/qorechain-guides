# Guvenlik Politikasi / Security Policy

Bu depo, QoreChain Guides dokumantasyon icerikleri icin topluluk tarafindan hazirlanan bagimsiz bir kaynaktir.

This repository is an independent community-maintained source for QoreChain Guides documentation.

## Kapsam / Scope

Bu guvenlik politikasi, depodaki dokumantasyon icerikleri, komut ornekleri, linkler ve rehber akislarini kapsar.

This security policy covers documentation content, command examples, links, and guide flows in this repository.

QoreChain protokolu, ag altyapisi, resmi servisler veya urun guvenligi bu deponun kapsami disindadir. Bu konular icin resmi QoreChain kanallari takip edilmelidir.

QoreChain protocol, network infrastructure, official services, or product security are outside the scope of this repository. Official QoreChain channels should be used for those topics.

## Bildirilebilecek Konular / What To Report

- Riskli veya yanlis komut onerileri
- Guvenli olmayan kurulum adimlari
- Supheli veya hatali dis linkler
- Gizli bilgi paylasmaya yonlendiren metinler
- Yanlis guvenlik varsayimlari

- Risky or incorrect command suggestions
- Unsafe setup steps
- Suspicious or incorrect external links
- Text that asks users to share secrets
- Incorrect security assumptions

## Gizli Bilgi Paylasmayin / Do Not Share Secrets

Issue, pull request veya yorumlarda su bilgileri paylasmayin:

Do not share the following in issues, pull requests, or comments:

- Seed phrase
- Private key
- API key
- Sunucu sifresi / Server password
- Erisim tokenlari / Access tokens
- Kisisel veya hassas bilgiler / Personal or sensitive information

## Bildirim Yolu / Reporting Flow

Dokumantasyon kaynakli bir guvenlik sorunu fark ederseniz, mumkunse hassas bilgi eklemeden issue acin ve problemi kisaca aciklayin.

If you notice a documentation-related security issue, open an issue when possible without including sensitive information, and briefly explain the problem.

Kritik veya resmi QoreChain guvenligiyle ilgili konularda resmi QoreChain kaynaklari ve duyurulari takip edilmelidir.

For critical or official QoreChain security topics, official QoreChain sources and announcements should be followed.
