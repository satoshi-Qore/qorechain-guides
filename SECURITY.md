# Güvenlik Politikası / Security Policy

Bu depo, QoreChain Guides dokümantasyon içerikleri için topluluk tarafından hazırlanan bağımsız bir kaynaktır.

This repository is an independent community-maintained source for QoreChain Guides documentation.

## Kapsam / Scope

Bu güvenlik politikası, depodaki dokümantasyon içerikleri, komut örnekleri, linkler ve rehber akışlarını kapsar.

This security policy covers documentation content, command examples, links, and guide flows in this repository.

QoreChain protokolü, ağ altyapısı, resmi servisler veya ürün güvenliği bu deponun kapsamı dışındadır. Bu konular için resmi QoreChain kanalları takip edilmelidir.

QoreChain protocol, network infrastructure, official services, or product security are outside the scope of this repository. Official QoreChain channels should be used for those topics.

## Bildirilebilecek Konular / What To Report

- Riskli veya yanlış komut önerileri
- Güvenli olmayan kurulum adımları
- Şüpheli veya hatalı dış linkler
- Gizli bilgi paylaşmaya yönlendiren metinler
- Yanlış güvenlik varsayımları

- Risky or incorrect command suggestions
- Unsafe setup steps
- Suspicious or incorrect external links
- Text that asks users to share secrets
- Incorrect security assumptions

## Gizli Bilgi Paylaşmayın / Do Not Share Secrets

Issue, pull request veya yorumlarda şu bilgileri paylaşmayın:

Do not share the following in issues, pull requests, or comments:

- Seed phrase
- Private key
- API key
- Sunucu şifresi / Server password
- Erişim tokenları / Access tokens
- Kişisel veya hassas bilgiler / Personal or sensitive information

## Bildirim Yolu / Reporting Flow

Dokümantasyon kaynaklı bir güvenlik sorunu fark ederseniz, mümkünse hassas bilgi eklemeden issue açın ve problemi kısaca açıklayın.

If you notice a documentation-related security issue, open an issue when possible without including sensitive information, and briefly explain the problem.

Kritik veya resmi QoreChain güvenliğiyle ilgili konularda resmi QoreChain kaynakları ve duyuruları takip edilmelidir.

For critical or official QoreChain security topics, official QoreChain sources and announcements should be followed.
