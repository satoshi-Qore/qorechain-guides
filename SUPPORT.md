# Destek / Support

Bu sayfa, QoreChain Guides deposu ile ilgili destek ve geri bildirim akışlarını açıklar.

This page explains support and feedback flows for the QoreChain Guides repository.

## Ne İçin Issue Açabilirsiniz? / What Can You Open an Issue For?

- Eksik veya belirsiz dokümantasyon
- Kurulum adımlarında açık olmayan bölümler
- Light Node operasyon notları
- Hata veya log açıklamaları
- Türkçe veya İngilizce çeviri önerileri
- Link, tablo veya format düzeltmeleri

- Missing or unclear documentation
- Setup steps that need clarification
- Light Node operation notes
- Error or log explanations
- Turkish or English translation suggestions
- Link, table, or formatting fixes

## Issue Şablonları / Issue Templates

Yeni issue açarken uygun şablonu seçin:

Choose the relevant template when opening a new issue:

- Documentation update / Dokümantasyon güncellemesi
- Troubleshooting report / Sorun giderme bildirimi

## Neleri Paylaşmayın? / What Not To Share

Aşağıdaki bilgileri issue, pull request veya yorumlarda paylaşmayın:

Do not share the following information in issues, pull requests, or comments:

- Seed phrase
- Private key
- API key
- Sunucu şifresi / Server password
- Tam erişim tokenleri / Full-access tokens
- Kişisel veya hassas bilgiler / Personal or sensitive information

## Resmi Kaynak Notu / Official Source Note

Bu depo topluluk tarafından hazırlanan bağımsız bir bilgi merkezidir. Kritik ağ, stake, güvenlik veya resmi duyuru konularında resmi QoreChain kaynakları kontrol edilmelidir.

This repository is an independent community-maintained knowledge hub. Official QoreChain sources should be checked for critical network, stake, security, or announcement-related topics.
