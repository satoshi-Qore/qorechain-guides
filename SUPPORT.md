# Destek / Support

Bu sayfa, QoreChain Guides deposu ile ilgili destek ve geri bildirim akislarini aciklar.

This page explains support and feedback flows for the QoreChain Guides repository.

## Ne Icin Issue Acabilirsiniz? / What Can You Open an Issue For?

- Eksik veya belirsiz dokümantasyon
- Kurulum adimlarinda acik olmayan bolumler
- Light Node operasyon notlari
- Hata veya log aciklamalari
- Turkce veya Ingilizce ceviri onerileri
- Link, tablo veya format duzeltmeleri

- Missing or unclear documentation
- Setup steps that need clarification
- Light Node operation notes
- Error or log explanations
- Turkish or English translation suggestions
- Link, table, or formatting fixes

## Issue Sablonlari / Issue Templates

Yeni issue acarken uygun sablonu secin:

Choose the relevant template when opening a new issue:

- Documentation update / Dokümantasyon guncellemesi
- Troubleshooting report / Sorun giderme bildirimi

## Neleri Paylasmayin? / What Not To Share

Asagidaki bilgileri issue, pull request veya yorumlarda paylasmayin:

Do not share the following information in issues, pull requests, or comments:

- Seed phrase
- Private key
- API key
- Sunucu sifresi / Server password
- Tam erisim tokenlari / Full-access tokens
- Kisisel veya hassas bilgiler / Personal or sensitive information

## Resmi Kaynak Notu / Official Source Note

Bu depo topluluk tarafindan hazirlanan bagimsiz bir bilgi merkezidir. Kritik ag, stake, güvenlik veya resmi duyuru konularinda resmi QoreChain kaynaklari kontrol edilmelidir.

This repository is an independent community-maintained knowledge hub. Official QoreChain sources should be checked for critical network, stake, security, or announcement-related topics.
