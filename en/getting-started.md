# Getting Started with QoreChain

Welcome to the QoreChain community documentation hub. This guide helps new participants understand the QoreChain ecosystem and find the right starting point.

## What is QoreChain?

QoreChain is a Layer 1 blockchain designed with post-quantum cryptography at its core. It is compatible with EVM (Ethereum Virtual Machine), CosmWasm, and SVM (Solana Virtual Machine), allowing developers to build on familiar tooling while benefiting from next-generation security.

Key properties:
- Post-quantum resistant cryptography
- Multi-VM compatibility (EVM / CosmWasm / SVM)
- Community-governed network
- Light Node participation model — no heavy hardware required

## How to Participate

There are several ways to get involved in QoreChain:

### Run a Light Node

Light Node operators help maintain network availability. Requirements are modest — a small VPS is enough.

Start here: [Light Node Operations](./light-node-operations.md)

### Contribute Documentation

This repository is community-maintained. If you spot an error, want to add a guide, or want to translate existing content, contributions are welcome.

### Translate Content

The repository is bilingual (English / Turkish). If you want to help translate a page, open a translation request issue.

Start here: [Translation Request Template](../.github/ISSUE_TEMPLATE/translation-request.md)

### Request Content

## Prerequisites

To run a Light Node, you need:

| Requirement | Minimum |
|---|---|
| Operating System | Ubuntu 20.04+ |
| CPU | 2 cores |
| RAM | 4 GB |
| Disk | 50 GB SSD |
| Docker | Installed |
| Docker Compose | Installed |
| Stake | 1000 QOR |

If you are only contributing documentation, no special hardware is required — just a GitHub account.
