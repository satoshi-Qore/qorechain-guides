# Getting Started with QoreChain

Welcome to the QoreChain community documentation hub. This guide helps new participants understand the ecosystem and choose the right starting point.

## What Is QoreChain?

QoreChain is a Layer 1 blockchain ecosystem focused on post-quantum security, multi-VM compatibility, and community-accessible infrastructure participation.

The ecosystem is commonly discussed around these areas:

- post-quantum cryptography;
- EVM, CosmWasm, and SVM compatibility;
- Light Node participation;
- community documentation and onboarding;
- developer and operator education.

This page is a community-maintained introduction. It does not replace official QoreChain documentation or announcements.

## Basic Requirements for Light Node Operators

Before running a Light Node, review the latest official instructions and confirm that your environment is suitable.

| Requirement | Suggested Baseline |
|---|---|
| Operating system | Ubuntu 22.04+ |
| CPU | 2 cores or more |
| RAM | 4 GB or more |
| Disk | 50 GB SSD or more |
| Docker | Installed |
| Docker Compose | Installed |
| Network access | Required ports open according to current instructions |

Use [Light Node Operations](./light-node-operations.md) for the full operational guide.

## Common First Steps

- Read the FAQ before asking repeated questions.
- Learn the basic terms in the glossary.
- Use the Light Node guide only after checking current official instructions.
- Keep screenshots and proof links free of sensitive information.
- Do not share private keys, seed phrases, API keys, or server credentials.
- Treat mainnet, validator, staking, and reward-related details as update-sensitive.

## Troubleshooting

If you encounter setup or panel access problems, start with the [Troubleshooting Guide](./troubleshooting.md). It covers common Light Node issues such as panel access, containers, logs, updates, and resource checks.
