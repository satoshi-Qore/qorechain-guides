# Eigenstate 2 Education Tasks

This guide covers the Education category in QoreChain Eigenstate 2.

Observed category size: 8 tasks.

Education tasks usually involve reading, watching, completing quizzes, bookmarking official resources, setting wallpapers, and learning core QoreChain concepts.

## Purpose

The goal of Education tasks is to help users understand QoreChain before participating more deeply in community, developer, or on-chain activities.

## Common Task Types

| Task Type | Typical Proof | Notes |
|---|---|---|
| Read documentation | Source link and short completion note | Some tasks may be verified by quiz |
| Complete quiz or trivia | Task panel context and short note | Do not share private answers if prohibited |
| Watch explainer video | Video or channel link and short note | Use the requested official video when specified |
| Bookmark website | Screenshot proof or proof page link | Hide unrelated personal browser data |
| Set wallpaper | Screenshot proof or proof page link | Hide private desktop information if needed |
| Referral tasks | Official referral or task panel proof | Avoid spam or fake referrals |

## Recommended Workflow

1. Complete the learning action honestly.
2. Keep the official source link open.
3. If the task asks for a quiz, complete it through the official task flow.
4. If proof is needed, use a screenshot link or proof note.
5. Write a short description of what you completed.

## Description Examples

```text
I read the requested QoreChain document and completed the related quiz for manual review.
```

```text
I watched the requested QoreChain explainer video and submitted the official video/channel link.
```

```text
I bookmarked qorechain.io and provided a public proof link for review.
```

## Risk Notes

- Do not submit empty proof for screenshot-based tasks.
- Do not expose browser passwords, email, or wallet data in screenshots.
- Do not claim quiz completion if it was not completed.
