# Eigenstate 2 Social Tasks

This guide covers the Social category in QoreChain Eigenstate 2.

Observed category size: 79 tasks.

Social tasks usually involve public community activity on platforms such as X, Reddit, LinkedIn, Medium, YouTube, Telegram, and similar channels.

## Purpose

The goal of Social tasks is to measure useful community participation, not repeated low-quality posting. Good submissions should be clear, public when possible, and easy for reviewers to verify.

## Common Task Types

| Task Type | Typical Proof | Notes |
|---|---|---|
| Follow or subscribe | Profile/page link, screenshot proof if needed | Use the official page when available |
| Like or react | Public post link, screenshot proof if no link is generated | Likes often do not create unique proof links |
| Comment or reply | Direct comment/post link | Keep the comment relevant and non-spammy |
| Create a post | Public post link | Avoid exaggerated reward, price, or guarantee language |
| Share or repost | Shared post link | Make sure the post is visible to reviewers |
| Join a community | Community page link, profile proof, or screenshot | Avoid private invite links when possible |

## Recommended Workflow

1. Open the task and read the exact instruction.
2. Confirm the platform and account you will use.
3. Complete the social action once.
4. Copy the direct post/profile/page link if available.
5. If no proof link is created, use a public proof page or screenshot link.
6. Write a short description explaining what you did.

## Writing Quality

Good social posts should be:

- short and natural;
- related to QoreChain or the requested topic;
- free of investment promises;
- free of unsupported mainnet, reward, or price claims;
- different enough from previous posts to avoid looking automated.

## Description Examples

```text
I followed the requested QoreChain social page and submitted the public page link for manual review.
```

```text
I replied to the QoreChain post with a relevant comment and submitted the direct post link for review.
```

```text
I created a public post about QoreChain and submitted the post link as proof.
```

## Risk Notes

- Do not use bots or mass-like behavior.
- Do not copy the same comment across many posts.
- Do not claim approval before the task is reviewed.
- Do not share private account data or wallet secrets.
