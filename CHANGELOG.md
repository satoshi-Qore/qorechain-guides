## Değişiklik Türleri / Change Types

| Etiket / Label | Anlam / Meaning |
|---|---|
| `docs(en):` | İngilizce içerik değişikliği / English content change |
| `docs(tr):` | Türkçe içerik değişikliği / Turkish content change |
| `docs:` | Dil-bağımsız veya çift dilli değişiklik / Language-neutral or bilingual change |
| `fix:` | Kırık link, hatalı komut veya yazım düzeltmesi / Broken link, wrong command, or typo fix |
| `feat:` | Yeni bölüm veya sayfa eklenmesi / New section or page added |
| `refactor:` | Yapısal yeniden düzenleme, içerik değişikliği yok / Structural reorganization, no content change |
| `chore:` | Şablon, CI veya meta-dosya değişiklikleri / Template, CI, or meta-file changes |
