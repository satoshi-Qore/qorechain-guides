# Contributing to QoreChain Guides

Thank you for helping improve QoreChain Guides. This repository is a community-maintained documentation hub for technical resources, node operations, task workflows, and troubleshooting notes.

## What You Can Contribute

- Setup improvements
- Light Node operation notes
- Troubleshooting steps
- Frequently asked questions
- Turkish or English translations
- Corrections for unclear or outdated documentation
- New task workflow notes

## Documentation Style

- Keep steps short, practical, and easy to verify.
- Use tables for checks, statuses, and troubleshooting flows.
- Use fenced code blocks for commands.
- Avoid speculation. If a rule may change, point users to official sources.
- Keep Turkish and English pages aligned when possible.
- Use plain language for community readers.

## Bilingual Structure

Turkish pages live under `tr/`.

English pages live under `en/`.

When adding a new topic, try to add or plan both language versions. If only one language is ready, mention that the matching page is still planned.

## Suggested Workflow

1. Open an issue for large changes or new topics.
2. Keep pull requests focused on one topic.
3. Explain what changed and why.
4. Check links before submitting.
5. Make sure commands are placed in the correct context.

## Safety Note

This repository does not replace official QoreChain documentation. For critical setup, stake, network, or security steps, users should check official sources and current announcements.
