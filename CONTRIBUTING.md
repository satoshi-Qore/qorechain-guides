# QoreChain Guides Katkı Rehberi / Contributing Guide

QoreChain Guides'i geliştirmeye yardım ettiğiniz için teşekkürler. Bu depo; teknik kaynaklar, node operasyonları, görev süreçleri ve sorun giderme notları için topluluk tarafından hazırlanan bir dokümantasyon merkezidir.

Thank you for helping improve QoreChain Guides. This repository is a community-maintained documentation hub for technical resources, node operations, task workflows, and troubleshooting notes.

## Neler Katkı Olarak Eklenebilir?

- Kurulum iyileştirmeleri
- Light Node operasyon notları
- Sorun giderme adımları
- Sık sorulan sorular
- Türkçe veya İngilizce çeviriler
- Belirsiz ya da güncelliğini yitirmiş dokümantasyon düzeltmeleri
- Yeni görev akışı notları

## What You Can Contribute

- Setup improvements
- Light Node operation notes
- Troubleshooting steps
- Frequently asked questions
- Turkish or English translations
- Corrections for unclear or outdated documentation
- New task workflow notes

## Dokümantasyon Stili

- Adımları kısa, uygulanabilir ve kontrol edilebilir tutun.
- Kontroller, durumlar ve sorun giderme akışlarında tablo kullanın.
- Komutlar için kod blokları kullanın.
- Tahminden kaçının. Bir kural değişebilecekse kullanıcıları resmi kaynaklara yönlendirin.
- Mümkün olduğunda Türkçe ve İngilizce sayfaları aynı yapıda tutun.
- Topluluk okuyucuları için sade bir dil kullanın.

## Documentation Style

- Keep steps short, practical, and easy to verify.
- Use tables for checks, statuses, and troubleshooting flows.
- Use fenced code blocks for commands.
- Avoid speculation. If a rule may change, point users to official sources.
- Keep Turkish and English pages aligned when possible.
- Use plain language for community readers.

## İki Dilli Yapı

Türkçe sayfalar `tr/` klasörü altında yer alır.

İngilizce sayfalar `en/` klasörü altında yer alır.

Yeni bir konu eklerken mümkünse iki dildeki karşılığını da ekleyin veya planlayın. Sadece bir dil hazırsa, diğer dildeki sayfanın daha sonra ekleneceğini belirtin.

## Bilingual Structure

Turkish pages live under `tr/`.

English pages live under `en/`.

When adding a new topic, try to add or plan both language versions. If only one language is ready, mention that the matching page is still planned.

## Önerilen Akış

1. Büyük değişiklikler veya yeni konular için önce issue açın.
2. Pull request'leri tek bir konuya odaklı tutun.
3. Ne değiştiğini ve neden değiştiğini açıklayın.
4. Göndermeden önce linkleri kontrol edin.
5. Komutların doğru bağlamda verildiğinden emin olun.

## Suggested Workflow

1. Open an issue for large changes or new topics.
2. Keep pull requests focused on one topic.
3. Explain what changed and why.
4. Check links before submitting.
5. Make sure commands are placed in the correct context.

## Güvenlik Notu

Bu depo resmi QoreChain dokümantasyonunun yerine geçmez. Kritik kurulum, stake, ağ veya güvenlik adımları için kullanıcılar resmi kaynakları ve güncel duyuruları kontrol etmelidir.

## Safety Note

This repository does not replace official QoreChain documentation. For critical setup, stake, network, or security steps, users should check official sources and current announcements.
